My Wordpress Portfolio

For this Wordpress project, I was able to make a porfolio of my 3 project I completed in my SMAD 203 Interactive Design class. These three projects were...

  • Project 1: User Research

    For this project, I was tasked with coming up with a design question that I wanted to solve. I was able to conduct research and interviews with my fellow classmates and create user narratives and personas from my findings.

  • Project 2: Information Architecture

    This project consisted of using my prior research to continue to form a way to solve the design question I had. I conducted research on competitors of this type of design and created a wireframe of the app I wanted to make.

  • Project 3: User Narratives

    For my user narrative project, I used my prior research to create a list of user stories and needs. With that in mind, I was able to create narratives of the users that would benefit from my app and talk about their lifestyle and issues.
